# Instructions - Prochaine Étape

**Objectif :** Créer les entités de base pour la gestion des matchs de padel.

---

### **Contexte**

Maintenant que la sécurité est fonctionnelle, nous allons construire le cœur de l'application. Nous allons créer une entité `Match` pour stocker les informations sur les matchs.

### **Prochaines Actions (réalisées par l'assistant)**

1.  **Créer l'entité `Match.java`** dans le package `padeJeux1.com.model`.
2.  **Créer le `MatchRepository.java`** dans le package `padeJeux1.com.repository`.
3.  **Créer le `MatchController.java`** dans le package `padeJeux1.com.controller` pour exposer les endpoints de l'API pour les matchs.
